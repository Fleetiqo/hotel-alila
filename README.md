# Hôtel Alila - Website

Modern, responsive website for Hôtel Alila in Morin-Heights, Laurentides, Quebec.

🌐 **Live Site**: [View on GitHub Pages](https://YOUR_USERNAME.github.io/hotel-alila/)

## Features

- ✨ Modern, rustic design with earthy color palette
- 📱 Fully responsive (mobile, tablet, desktop)
- 🏨 10 different room types with high-quality photos
- 📍 Interactive location section with map
- 🖼️ Beautiful photo gallery with actual hotel images
- 📧 Contact form with reservation options
- 🌐 Bilingual ready (French/English)

## Room Types

1. **Latonia** - 1 lit Queen
2. **Labelle** - 1 lit King
3. **Ladonna** - 2 lits Queen
4. **Latisha** - 2 lits Queen avec plancher de bois franc
5. **Kimila** - 1 lit King avec foyer au gaz propane
6. **Kingston** - 1 lit King, foyer, plancher de bois franc
7. **Oméga** - 1 lit King, bain tourbillon, plancher de bois franc
8. **Tropicana** - 1 lit King, bain tourbillon, foyer, plancher de bois franc
9. **Yamaska** - Suite Junior
10. **Romantica** - Suite de Luxe

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
